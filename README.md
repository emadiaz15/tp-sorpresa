## Intrucciones

- Clonar repositorio
- Se recomienda instalar un entorno virtual
- Instalar los requerimientos
'''
$ pip install -r requirements.txt
'''
- Buildear la aplicacion
'''
docker-compose build
'''
- Runear la app
'''
docker-compose up -d
'''
- Para parar la aplicacion
'''
docker-compose down
'''
